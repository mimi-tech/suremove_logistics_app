# Suremove Logistics
This project is a delivery app built with flutter and bloc for state management using the<strong>MVVM architecture</strong> .
The APIs have not been connected yet only the screens.At this stage I have worked on all the UI. I am now about writing the logic for the app to be functioning.

## Getting Started

<ul>
<li>Clone the project</li>
<li>On the terminal type <strong>pub get</strong> to get all the dependencies</li>
<li>After you have done pub get, run <strong>flutter run</strong> This will run this project to either your simulator or emulator</li>
</ul>
